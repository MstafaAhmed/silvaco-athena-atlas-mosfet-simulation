# Silvaco ATHENA/ATLAS NMOS MOSFET Simulation

A complete TCAD workflow for a poly-gate, self-aligned NMOS transistor: process
fabrication in **ATHENA**, electrical characterization in **ATLAS**, and
post-processing/plotting in **MATLAB**.


## 1. Process Flow (ATHENA)

The device is built as a classic self-aligned poly-gate NMOS on a p-type
substrate:

| Step | Description            | Key parameters |
|---|              ---         |                        ---                      |
| 1 | Substrate initialization | Boron doping = 1.0e15 cm⁻³, `<100>` orientation |
| 2 | Field oxide growth | Dry O₂, 1250 °C, 250 min |
| 3 | Field oxide strip | Full etch, exposes bare silicon |
| 4 | Gate oxide growth | Dry O₂, 1000 °C, 50 min |
| 5 | Polysilicon gate deposition | Thickness = 0.2 µm |
| 6 | Gate patterning | Etch defines gate edge at x = 0.35 µm |
| 7 | Poly reoxidation | Dry O₂, 1000 °C, 10 min |
| 8 | Source/drain implant | Arsenic, dose = 5.0e14 cm⁻², energy = 50 keV, 0° tilt |
| 9 | Contact windows | Oxide etched over source/gate regions |
| 10 | Metallization | Aluminum, thickness = 0.05 µm |
| 11 | Mirror | Structure mirrored about the gate centerline to complete a symmetric device |

Interface fixed oxide charge: **Qf = 3×10¹⁰ cm⁻²**.

Final structure is written to `mosfet.str` and visualized in TonyPlot.

## 2. Electrical Characterization (ATLAS)

- Gate contact modeled with `n.poly` work function.
- Physics models: CVT mobility, Boltzmann statistics, 2-carrier transport, T = 300 K.
- Newton solver with trap-assisted convergence aids (`trap`, `autonr`).

**Bias conditions:**
- Three gate bias points: V_GS = 1 V, 2 V, 5 V
- Drain sweep at each gate bias: V_DS = 0 → 3.3 V, step 0.3 V

This produces the family of output characteristics (I_D vs V_DS) shown above,
plus data for extracting threshold voltage, on-resistance, and other
figures of merit.

## 3. MATLAB Post-Processing

The `1- MOSFET Matlab` folder contains scripts that:
- Import the `.dat` files generated by ATLAS's `extract curve(...)` commands
- Plot linear- and log-scale I_D–V_DS families of curves
- Estimate linear-region on-resistance (R_on) from the low-V_DS slope

## How to Run

**With Silvaco TCAD access:**
```bash
deckbuild -run mosfet_athena_atlas.in
```
This runs both the ATHENA process flow and ATLAS electrical simulation in
sequence, producing `mosfet.str`, `mosfet1.log`/`mosfet2.log`/`mosfet3.log`,
and the extracted `idvd_vg*.dat` files.

**Without Silvaco access (MATLAB only):**
Sample `.dat` output files are included in this repo — open
`1- MOSFET Matlab/plot_mosfet_idvd.m` in MATLAB and run it directly to
reproduce the plots from the pre-generated data.

## Why This Matters

Self-aligned poly-gate processing (patterning the gate *before* the
source/drain implant) is the technique that made scaled MOSFETs
manufacturable: the poly gate itself acts as an implant mask, so the
source/drain edges align to the gate with no separate lithography step and
no risk of gate-to-junction misalignment. This project walks through that
exact sequence — oxidation, poly deposition/patterning, self-aligned
implant, contact formation — and pairs it with a full DC electrical
characterization to close the loop between process parameters and device
behavior.
